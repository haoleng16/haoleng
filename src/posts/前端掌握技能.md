## border

```
/* 粗细 | 样式 | 颜色 */
border: 2px solid red;
```

- **宽度 (Width)**：墙有多厚？（如 `1px`, `5px`, `thick`）。
- **样式 (Style)**：墙长什么样？
  - `solid`：实线（最常用）。
  - `dashed`：虚线（像剪切线）。
  - `dotted`：点线。
  - `double`：双实线。
- **颜色 (Color)**：墙是什么颜色？（如 `red`, `#000000`, `transparent`）。

## border-radius:18px

圆角半径；数值越大，边角就越圆；数值为0时，边角就是锐利的直角

## display

**display:flex**:弹性布局

- 作用：让盒子里的子元素能够自动对齐、等间距分布或自动拉伸

\*\*display:grid：网格布局

- 作用：像Excel表格一样，把页面划为行和列
- 场景：复杂的仪表盘界面布局

## box-shadow

阴影

```
box-shadow: 0 8px 24px rgba(0, 0, 0, 0.12);
```

- 0: **水平偏移**
  阴影在水平方向不移动。这意味着阴影会在元素的正后方水平居中
- 8px **垂直偏移 -v -shadow**
  阴影心爱姑娘下移动了8像素。这会让元素看起来会像是“浮”在页面上
- 24px **模糊距离-blur**
  这是阴影的模糊半径
- rgba **颜色+透明度**

## gap:24px

在容器中设置间距

```
.navbar-links { 
  display: flex;
  gap: 24px; 
}
```

flex:\*\*一维（要么横，要么纵）

## var(--accent)

设置css某个值为这个变量，这样可以统一设置这一个值不必一处一处修改
例如：

```
.navbar-links a:hover { color: var(--accent); /* 这里使用了那个变量 */ }
```

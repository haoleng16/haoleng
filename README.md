## agent
打算做一个简历分析多agent助手，主要有四个主agent

- 简历分析助手： 根据简历匹配对应岗位JD进行打分，并分析简历薄弱点、面试问题。根据对应薄弱点生成笔记链接进行推送，帮助面试者更快获取面试秘籍
- 润色工作经历助手： 针对简历对应岗位大数据搜索某聘对应JD进行润色
- 匹配岗位助手：匹配岗位要求直接获取某聘数据对面试者简历推送岗位
- 模拟面试助手：对应个人岗位做面试题私人定制（通过rag知识库快速获取对应岗位真实的面试题推送）
